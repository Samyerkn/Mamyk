# MAMYQ

Adaptive clothing platform with a sponsorship system.

##  About

MAMYQ is a social web platform that helps families purchase adaptive clothing or receive financial support from sponsors.

The platform allows users to:
- Buy adaptive T-shirts
- Submit help requests
- Sponsor families in need
- Track donation progress
- Manage orders and applications
